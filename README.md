# Maestro-hci-project-202526
Project title: Maestro  
Course: ECE_CK904 Interactive Technologies  
@ University of Patras  
This project was developed as a group assignment.  

Maestro is an interactive application that allows users to conduct music in real-time using hand gestures. By leveraging Computer Vision and Haptic Feedback, the user controls volume, tempo, and playback state while receiving synchronized tactile vibrations on bHaptics vests and sleeves.

## System Requirements:  
Webcam  
bHaptics Devices  
Python 3.10  
bHaptics Player Application  

### Requirements:  
```bash
pip install -r requirements.txt
```
